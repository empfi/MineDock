# Product

## Register

brand

## Users

Minecraft Java server owners who want direct local control without maintaining a web panel.

## Product Purpose

Show MineDock as the cleanest path from a local machine to a managed Minecraft server.

## Brand Personality

Precise, capable, energetic.

## Anti-references

No generic hosting-panel aesthetic, decorative grids, glass-card stacks, or loud gaming visuals.

## Design Principles

- Show the real product.
- Make local ownership the clearest message.
- Keep each section focused on one claim.
- Use the logo palette with restraint.

## Accessibility & Inclusion

WCAG AA contrast, keyboard focus, reduced-motion support.
